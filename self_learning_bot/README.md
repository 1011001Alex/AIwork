# 🤖 Самообучающийся Консольный Бот

**Бот, который становится ВАМИ!** Чем больше вы общаетесь, тем больше он перенимает ваш стиль общения, пока не станет вашей цифровой копией!

## ✨ Особенности

- **🧠 Самообучение**: Анализирует ваш стиль общения, словарный запас, использование эмодзи, частоту вопросов
- **🎯 Адаптация**: Постепенно перенимает вашу манеру речи, формальность/неформальность, длину ответов
- **💾 Память**: Сохраняет всю историю разговоров и паттерны поведения
- **⚙️ CLI Агент**: Может выполнять любые команды в терминале через `/exec`
- **🔄 Авто-дообучение**: Автоматически дообучает модель каждые N диалогов
- **📊 Профилирование**: Строит детальный профиль вашего стиля общения
- **⚡ Мгновенный запуск**: Работает без загрузки тяжелых нейросетей, использует умные паттерны

## 🚀 Быстрый старт

### 1. Запуск бота

```bash
cd self_learning_bot
python bot.py
```

**Всё!** Никаких дополнительных установок не требуется для базовой работы!

### 2. (Опционально) Установка зависимостей для продвинутого режима

Если хотите использовать нейросеть GPT-2 для более точной генерации:

```bash
pip install -r requirements.txt
```

Затем отредактируйте `config/config.json` и установите `"skip_model_loading": false`

## 📋 Команды

| Команда | Описание |
|---------|----------|
| `/stats` | Показать статистику бота и модели |
| `/profile` | Показать ваш профиль и стиль общения |
| `/train` | Принудительно запустить дообучение |
| `/clear` | Очистить контекст разговора |
| `/exec <cmd>` | Выполнить системную команду |
| `/help` | Показать справку |
| `/quit` | Сохранить прогресс и выйти |

## 🔧 Как это работает

### Архитектура

```
self_learning_bot/
├── bot.py                 # Главный файл бота
├── config/
│   └── config.json        # Конфигурация
├── memory/
│   ├── memory_manager.py  # Управление памятью и профилирование
│   └── conversation_history.json  # История разговоров (создается автоматически)
├── models/
│   └── adaptive_model.py  # Адаптивная языковая модель
├── utils/
│   └── command_executor.py # Выполнение команд
└── requirements.txt       # Зависимости (опционально)
```

### Процесс обучения

1. **Сбор данных**: Каждый диалог сохраняется в памяти
2. **Анализ паттернов**: Извлечение стиля, тем, словарного запаса
3. **Классификация**: Определение типа сообщения (приветствие, вопрос, код, технологии, творчество)
4. **Адаптация**: Генерация ответа с учётом вашего стиля

### Что анализируется

- ✅ Средняя длина ответов
- ✅ Использование эмодзи
- ✅ Частота вопросов
- ✅ Формальность/неформальность
- ✅ Тематика разговоров (код, технологии, творчество, личное)
- ✅ Уникальный словарный запас
- ✅ Паттерны команд

### Категории ответов

Бот распознаёт следующие категории сообщений:

- **Приветствия** (`привет`, `здравствуй`, `хай`)
- **Прощания** (`пока`, `до свидания`)
- **Вопросы** (содержат `?` или вопросительные слова)
- **Программирование** (`код`, `python`, `функция`, `баг`)
- **Технологии** (`ИИ`, `нейросеть`, `сервер`, `алгоритм`)
- **Творчество** (`креатив`, `дизайн`, `идея`)
- **Команды** (начинаются с `/` или содержат `выполни`)

## 🎛️ Конфигурация

Отредактируйте `config/config.json`:

```json
{
  "model_name": "gpt2",           // Модель для использования
  "max_length": 512,              // Максимальная длина ответа
  "temperature": 0.8,             // Креативность (0-1)
  "top_p": 0.95,                  // Top-p sampling
  "learning_rate": 1e-4,          // Скорость обучения
  "training_epochs": 3,           // Эпох обучения
  "batch_size": 4,                // Размер батча
  "min_samples_for_training": 10  // Мин. диалогов для обучения
}
```

## 🎯 Долгосрочная цель

После достаточного количества диалогов (рекомендуется 50-100+):

- Бот будет отвечать **точно как вы**
- Использовать **ваши любимые слова и фразы**
- Поддерживать **ваш типичный тон и стиль**
- Понимать **ваши интересы и темы**
- Стать вашей **персональной AI-копией**!

## ⚠️ Важные заметки

- Первое дообучение займёт несколько минут
- Для лучшего результата общайтесь на разные темы
- Модель сохраняет прогресс между сессиями
- Команды проходят проверку безопасности

## 🛠️ Расширение

Вы можете легко расширить функционал:

- Добавить новые анализаторы паттернов в `memory_manager.py`
- Интегрировать другие модели в `adaptive_model.py`
- Добавить новые команды в `bot.py`
- Расширить проверки безопасности в `command_executor.py`
